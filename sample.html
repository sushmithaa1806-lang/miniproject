Abstract Detail

The main idea of this project is to develop a Smart Traffic Management System that uses AI and Computer Vision to control traffic lights dynamically based on real-time traffic conditions. Instead of following fixed signal timings, the system analyzes the live camera feed to detect the number of vehicles in each lane and automatically adjusts signal durations. Additionally, it identifies emergency vehicles like ambulances and fire trucks and provides green signal priority, ensuring they can move smoothly without getting stuck in traffic. The solution of our project uses CCTV cameras and computer vision models to detect and count vehicles in real time.
